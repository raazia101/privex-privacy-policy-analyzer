🔧 Troubleshooting

  • Extension not loading? Check Chrome Extensions for errors, reload extension
  • No automatic detection? Try refreshing the page, check console logs
  • API errors? Ensure Flask server is running on http://127.0.0.1:5000
  • CORS issues? Check that flask-cors is properly configured
